# Referral Tracking System

## Project Overview
The Referral Tracking System is designed to help businesses manage and analyze their referral programs. It provides comprehensive tools for tracking referrals, implementing incentive programs, and analyzing performance metrics.

## Features
- **Referral Management**: Easily create and manage referral campaigns.
- **Analytics Dashboard**: Monitor performance metrics such as referral sources, conversion rates, and user engagement.
- **Incentive Programs**: Offer rewards to users for each successful referral.
- **User-Friendly Interface**: Intuitive design for ease of use.

## Technology Stack
- **Frontend**: React.js
- **Backend**: Node.js and Express
- **Database**: MongoDB
- **Deployment**: AWS

## Setup Instructions
1. **Clone the Repository**:
    ```bash
    git clone https://github.com/Raunakmali08/referral-tracking-system.git
    cd referral-tracking-system
    ```  
2. **Install Dependencies**:
    ```bash
    npm install
    ```  
3. **Configure Environment Variables**: Create a `.env` file in the root directory and add your configuration settings.

4. **Run the Application**:
    ```bash
    npm start
    ```  
5. **Access the Application**: Open your browser and go to `http://localhost:3000`.

## Running Tests
To run tests, use the command:
```bash
npm test
```
